# Contributing

Contributions are welcome! This project helps automate DNS record management between Traefik and Pi-hole.

## Before You Submit

Please ensure:
- [ ] Code follows existing style and patterns
- [ ] No hardcoded personal domains or IP addresses
- [ ] Updated README if adding new features
- [ ] Tested changes locally

## Environment-Specific Data

**Never commit:**
- Your actual `.env` file
- Real domain names (use `example.com`)
- Real IP addresses (use `192.168.1.x` or `10.0.0.x`)
- Passwords or tokens

Use `.env.example` as a template with generic placeholders.

## Pull Request Process

1. Fork the repository
2. Create a feature branch
3. Make your changes
4. Test thoroughly
5. Submit a PR with a clear description
